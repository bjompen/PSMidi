# PSMidi playground

**IMPORTANT NOTE:**
- Before anything here works you _will_ need the module included in the MIDI package below.
- This module is very much a beta. Maybe even Alpha. things may not work. Drop a bug report if they don't

---

installation:
- Manually:
  - Download, build, import
- From PowerShell Gallery
  - Find-Module PSMidi | Install-Module

---

Midi 2.0 is coming to Windows 11 - and you can read all about that [here](https://microsoft.github.io/MIDI/)

This module adds functionality from the built in Midi module `WindowsMidiServices` that currently comes with the MIDI install package above. More fun stuff to come.
